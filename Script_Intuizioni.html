Step 2: Analisi delle Intuizioni

Questo passaggio ti guida nell'uso dello script per analizzare i testi selezionati, identificare connessioni intuitive e raggruppare contenuti significativi.

Script Operativo

Lo script seguente ti permette di analizzare i testi raccolti, evidenziando intuizioni e connessioni inattese.


# Script per Analisi Intuizioni

def analyze_intuitive_connections(texts):
    \"\"\"
    Analizza i testi raccolti per individuare connessioni e intuizioni.
    
    Args:
    - texts (list of str): Lista di testi da analizzare.
    
    Returns:
    - dict: Cluster di contenuti organizzati per connessioni intuitive.
    \"\"\"
    from sklearn.feature_extraction.text import TfidfVectorizer
    from sklearn.cluster import KMeans
    
    # Trasforma i testi in vettori TF-IDF
    vectorizer = TfidfVectorizer(stop_words='english')
    X = vectorizer.fit_transform(texts)
    
    # Applica clustering K-Means
    num_clusters = min(len(texts), 5)  # Massimo 5 cluster
    kmeans = KMeans(n_clusters=num_clusters, random_state=42)
    kmeans.fit(X)
    
    # Raggruppa i testi per cluster
    clusters = {i: [] for i in range(num_clusters)}
    for idx, label in enumerate(kmeans.labels_):
        clusters[label].append(texts[idx])
    
    return clusters

def summarize_clusters(clusters):
    \"\"\"
    Crea un riassunto dei cluster di intuizioni.
    
    Args:
    - clusters (dict): Cluster di contenuti.

    Returns:
    - str: Riassunto testuale dei cluster.
    \"\"\"
    summary = "Riassunto dei Cluster di Intuizioni:\\n"
    for cluster_id, texts in clusters.items():
        summary += f"Cluster {cluster_id + 1}:\\n"
        for text in texts:
            summary += f"- {text[:50]}...\\n"  # Mostra i primi 50 caratteri di ogni testo
        summary += "\\n"
    
    return summary

# Test dello script
texts_example = [
    "Il cielo è blu e l'erba è verde.",
    "I sogni sono finestre verso l'ignoto.",
    "La curiosità guida l'esplorazione dell'universo.",
    "I ricordi intrecciano passato e presente.",
    "La connessione tra mente e corpo è profonda."
]

clusters = analyze_intuitive_connections(texts_example)
summary = summarize_clusters(clusters)
print(summary)
