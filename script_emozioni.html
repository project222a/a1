Script e Rielaborazione delle Emozioni

Questa pagina fornisce lo script per analizzare le emozioni e guida il processo di rielaborazione. È il passaggio centrale del percorso emotivo.

1. Script di Analisi Emozionale

Lo script seguente ti permette di analizzare le emozioni predominanti nei testi. Copialo e usalo per iniziare il processo.


# Script per Analisi Emozionale

def analyze_emotions_in_text(content):
    import re
    emotion_pattern = r"E=([\w\s]+)\((\d+)\)"
    matches = re.findall(emotion_pattern, content)
    emotions = {emotion.strip(): int(intensity) for emotion, intensity in matches}
    return emotions

def generate_emotional_clusters(texts):
    clusters = {}
    for text in texts:
        emotions = analyze_emotions_in_text(text)
        for emotion, intensity in emotions.items():
            if emotion not in clusters:
                clusters[emotion] = []
            clusters[emotion].append((text, intensity))
    return clusters

def map_emotions_over_time(emotional_data):
    emotion_timeline = {}
    for entry in emotional_data:
        timestamp = entry.get("timestamp", "unknown")
        emotions = entry.get("emotions", {})
        for emotion, intensity in emotions.items():
            if emotion not in emotion_timeline:
                emotion_timeline[emotion] = []
            emotion_timeline[emotion].append({"timestamp": timestamp, "intensity": intensity})
    return emotion_timeline

def generate_emotional_summary(clusters, timeline):
    summary = "Riassunto Analisi Emozionale\\n"
    summary += "===========================\\n\\n"
    summary += "Cluster di Emozioni:\\n"
    for emotion, texts in clusters.items():
        summary += f"- {emotion.capitalize()}:\\n"
        for text, intensity in texts:
            summary += f"  * Intensità {intensity}: {text[:50]}...\\n"
        summary += "\\n"
    summary += "Mappa Temporale delle Emozioni:\\n"
    for emotion, entries in timeline.items():
        summary += f"- {emotion.capitalize()}:\\n"
        for entry in entries:
            summary += f"  * {entry['timestamp']}: Intensità {entry['intensity']}\\n"
        summary += "\\n"
    file_path = "/mnt/data/Riassunto_Emozionale.txt"
    with open(file_path, "w") as file:
        file.write(summary)
    return file_path

2. Rielaborazione delle Emozioni

Dopo aver completato l'analisi, segui questi passaggi per una riflessione più approfondita:

  1. Revisione: Esamina i risultati dell'analisi emozionale e identifica i punti chiave.
  2. Domande Guida: Utilizza queste domande per sviluppare nuove idee:
    • Quali sono i punti chiave emersi?
    • Quali connessioni o contraddizioni noti?
    • Quali nuove conclusioni puoi trarre?
  3. Riassunto: Scrivi un testo che sintetizzi le tue riflessioni.
  4. Archiviazione: Salva il riassunto nel sistema di memoria o in un archivio dedicato.
